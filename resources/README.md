# AgroGuard Mobil

Bu klasor Capacitor mobil paketleme icin kullanilir.

## Gerekli adimlar
1. Paketleri kur:
   npm install
2. Web build al:
   npm run build
3. iOS/Android projeleri ekle:
   npx cap add ios
   npx cap add android
4. Senkronize et:
   npx cap sync

## Notlar
- Splash screen ve ikonlar icin @capacitor/assets tavsiye edilir.
- default renkler: tema #0f3b23, arka plan #e7f7ed
